# User Interface Guide

## Overview

The Behind Bars mod includes several UI elements to help you understand your status and interact with the jail system.

## Jail Info UI

Shown when you're in jail, displays:

### Information Shown
- **Jail Time Countdown**: Real-time countdown of remaining sentence
- **Crime Information**: What crime you were arrested for
- **Bail Amount**: Current bail amount (updates in real-time)
- **Dynamic Updates**: All information updates as time progresses

### When It Appears
- Automatically shown when you're arrested
- Stays visible while in jail
- Updates continuously

### How to Use
- No interaction needed - just watch the countdown
- Check bail amount if you want to pay
- See how much time you have left

## Bail UI

Dedicated interface for bail payments:

### Features
- **Bail Amount Display**: Shows your calculated bail
- **Payment Options**: Choose to pay bail
- **Multiplayer Info**: Shows if friends can help pay
- **Negotiation**: Attempt to negotiate bail amount

### How to Access
- Automatically shown after arrest
- Can be accessed from Jail Info UI
- Available while in jail

### Using the Bail UI
1. Review your bail amount
2. Check if you can afford it
3. Choose to pay bail
4. Confirm payment
5. You'll be released immediately if successful

## Parole Status UI

Persistent UI showing your parole status:

### Information Displayed
- **Time Remaining**: How much parole time is left
- **LSI Level**: Your current risk assessment (Minimum, Medium, High, Severe)
- **Search Probability**: Your chance of being searched (percentage)
- **Violation Count**: Number of violations you've committed

### When It Appears
- Shown automatically when you're on parole
- Stays visible throughout parole period
- Updates in real-time

### Purpose
- Keep track of your parole status
- Know your search risk level
- Monitor violations

## Parole Conditions UI

Modal dialog shown when you're released on parole:

### Information Shown
- **Parole Duration**: How long you'll be on parole
- **Supervision Level**: Your LSI risk level
- **Prohibited Activities**: What you can't do
- **Search Conditions**: When and how you can be searched

### When It Appears
- Shown once when parole starts
- Appears after release from jail
- Must be acknowledged to dismiss

### Important Information
- Read carefully - these are your parole rules
- Violations have consequences
- Search conditions explain when officers can search you

## Wanted Level UI

Visual indicator of your wanted status:

### Display
- **Star System**: 0-5 stars showing wanted level
- **Visual Indicator**: Easy to see at a glance
- **Updates**: Changes as you commit crimes or get arrested

### What It Means
- **0 Stars**: Not wanted
- **1-2 Stars**: Minor crimes
- **3-4 Stars**: Serious crimes
- **5 Stars**: Most wanted

### Purpose
- Quick reference for your wanted status
- Helps you know if police are looking for you
- Affects how aggressively you're pursued

## Officer Command UI

Interface for officer commands and interactions:

### Features
- **Guard Commands**: Interact with jail guards
- **Officer Interactions**: Communicate with officers
- **System Commands**: Access jail system features

### When Available
- While in jail
- During booking process
- When interacting with officers

## Tips

- All UIs update in real-time
- Pay attention to parole conditions
- Monitor your wanted level
- Check bail amounts regularly if considering payment
- Parole Status UI helps you avoid violations
